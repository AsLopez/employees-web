## Getting Started

### Prerequisites

- npm version: `8.15.0`
- node version: `16.17.0`
- Angular version: `16.1.1`

#### Install dependencies

```sh
sh npm i
```

## Development

### Start local environment

```sh
npm start
```

### Test

npm run test
